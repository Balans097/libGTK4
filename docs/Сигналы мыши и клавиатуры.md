# Сигналы действий мыши и клавиатуры в GTK4

## 🖱️ Сигналы мыши

### Основные события мыши (GtkWidget)
| Сигнал | Когда вызывается | Параметры |
|--------|------------------|-----------|
| button-press-event | Нажата кнопка мыши | widget, event |
| button-release-event | Отпущена кнопка мыши | widget, event |
| enter-notify | Курсор вошёл в область виджета | widget, event |
| leave-notify | Курсор покинул область виджета | widget, event |
| motion-notify | Движение курсора | widget, event |
| scroll-event | Прокрутка колеса мыши | widget, event |

---

### Сигналы мыши в GtkButton / ToggleButton
| Сигнал | Когда вызывается | Параметры |
|--------|------------------|-----------|
| clicked | Клик по кнопке | button |
| pressed | Кнопка нажата | button |
| released | Кнопка отпущена | button |

---

### Сигналы мыши в GtkEntry
| Сигнал | Когда вызывается | Параметры |
|--------|------------------|-----------|
| icon-press | Клик по иконке | entry, icon_pos, event |
| icon-release | Отпущена иконка | entry, icon_pos, event |

---

### Сигналы мыши в GtkListBox
| Сигнал | Когда вызывается | Параметры |
|--------|------------------|-----------|
| row-activated | Клик/активация строки | list_box, row |
| child-activated | Клик по дочернему элементу | list_box, child |

---

### Сигналы мыши в GtkGesture
| Сигнал | Виджет | Когда вызывается |
|--------|--------|------------------|
| pressed | GtkGestureClick | Нажатие кнопки |
| released | GtkGestureClick | Отпускание кнопки |
| clicked | GtkGestureClick | Клик (press + release) |
| drag-begin | GtkGestureDrag | Начало перетаскивания |
| drag-update | GtkGestureDrag | Обновление перетаскивания |
| drag-end | GtkGestureDrag | Конец перетаскивания |
| swipe | GtkGestureSwipe | Свайп |
| long-pressed | GtkGestureLongPress | Долгое нажатие |
| rotate | GtkGestureRotate | Вращение (мультитач) |
| zoom | GtkGestureZoom | Масштабирование (мультитач) |
| enter | GtkEventControllerMotion | Курсор вошёл |
| leave | GtkEventControllerMotion | Курсор вышел |
| motion | GtkEventControllerMotion | Движение курсора |
| scroll | GtkEventControllerScroll | Прокрутка |

---

## ⌨️ Сигналы клавиатуры

### Основные события клавиатуры (GtkWidget)
| Сигнал | Когда вызывается | Параметры |
|--------|------------------|-----------|
| key-press-event | Нажата клавиша | widget, event |
| key-release-event | Отпущена клавиша | widget, event |

---

### Сигналы клавиатуры в GtkEntry
| Сигнал | Когда вызывается | Параметры |
|--------|------------------|-----------|
| activate | Нажат Enter | entry |
| preedit-changed | Изменён preedit-текст | entry |

---

### Сигналы клавиатуры в GtkTextView
| Сигнал | Когда вызывается | Параметры |
|--------|------------------|-----------|
| backspace | Нажат Backspace | text_view |
| insert-at-cursor | Вставка текста | text_view, string |
| move-cursor | Перемещение курсора | text_view, step, count, extend_selection |

---

### Сигналы клавиатуры в GtkButton / ToggleButton
| Сигнал | Когда вызывается | Параметры |
|--------|------------------|-----------|
| activate | Нажат Enter/Space | button |

---

### Сигналы клавиатуры в GtkGesture
| Сигнал | Виджет | Когда вызывается |
|--------|--------|------------------|
| key-pressed | GtkEventControllerKey | Нажата клавиша |
| key-released | GtkEventControllerKey | Отпущена клавиша |
